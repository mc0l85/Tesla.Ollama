To solve this problem, we need to merge two sorted lists of integers into a single sorted list without using any built-in sorting functions. The solution involves efficiently combining the elements from both lists while maintaining the sorted order.

### Approach
1. **Problem Analysis**: The task is to merge two sorted lists into one sorted list. The key insight is that since both input lists are already sorted, we can use a two-pointer technique to traverse both lists simultaneously, comparing elements at the current pointers, and appending the smaller element to the result list.
2. **Algorithm Selection**: 
   - Initialize two pointers, `i` for the first list and `j` for the second list.
   - Traverse both lists while comparing elements at the current pointers.
   - Append the smaller element to the result list and move the corresponding pointer forward.
   - Once one list is exhausted, append all remaining elements from the other list to the result.
3. **Complexity Analysis**: The algorithm runs in O(n + m) time, where n and m are the lengths of the two input lists. This is optimal as each element is processed exactly once. The space complexity is O(n + m) to store the merged result.

### Solution Code
```python
def merge_sorted(list1, list2):
    """
    Merge two sorted lists of integers into a single sorted list.
    
    Args:
        list1 (list): First sorted list of integers.
        list2 (list): Second sorted list of integers.
        
    Returns:
        list: A new sorted list containing all elements from both input lists.
        
    Example:
        >>> merge_sorted([1, 3, 5], [2, 4, 6])
        [1, 2, 3, 4, 5, 6]
    """
    i = 0
    j = 0
    merged = []
    
    while i < len(list1) and j < len(list2):
        if list1[i] <= list2[j]:
            merged.append(list1[i])
            i += 1
        else:
            merged.append(list2[j])
            j += 1
            
    while i < len(list1):
        merged.append(list1[i])
        i += 1
        
    while j < len(list2):
        merged.append(list2[j])
        j += 1
        
    return merged
```

### Explanation
1. **Initialization**: We start with two pointers `i` and `j` set to 0, pointing to the start of `list1` and `list2` respectively. An empty list `merged` is initialized to store the result.
2. **Merging Process**: 
   - While both pointers are within their respective list bounds, compare elements at `list1[i]` and `list2[j]`.
   - Append the smaller element to `merged` and increment the corresponding pointer.
3. **Appending Remaining Elements**: After one list is exhausted, append all remaining elements from the other list to `merged`.
4. **Return Result**: The merged list, now containing all elements in sorted order, is returned.
